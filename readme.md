## Quick start 
### Installation
1. install pytorch >= v1.4.0 following [official instruction](https://pytorch.org/).
    - **tested with pytorch v1.9.0**
2. install dependencies:
```
pip install -r requirements.txt
```
3. Testing
For test1.py:
```
python test1.py
```
